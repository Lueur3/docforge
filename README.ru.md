# DocForge

Графический интерфейс для [MarkItDown](https://github.com/microsoft/markitdown) и [Pandoc](https://pandoc.org/).
Конвертируйте документы между форматами без командной строки.

[English documentation](README.md)

---

## Возможности

### Вкладка MarkItDown

Конвертация любого поддерживаемого файла в Markdown с помощью библиотеки Microsoft MarkItDown.

Поддерживаемые входные форматы: Word, PDF, Excel, PowerPoint, HTML, изображения (с OCR) и другие.

- Путь к выходному файлу подставляется автоматически (та же папка, расширение `.md`)
- Опциональная поддержка аудио/видео через ffmpeg — установка одной кнопкой прямо в приложении

### Вкладка Pandoc

Полноценная двунаправленная конвертация форматов документов через Pandoc.

| Формат вывода | Расширение |
|---|---|
| Markdown | `.md` |
| HTML | `.html` |
| Word Document | `.docx` |
| EPUB | `.epub` |
| reStructuredText | `.rst` |
| Обычный текст | `.txt` |
| LaTeX | `.tex` |
| ODT | `.odt` |
| RTF | `.rtf` |
| PDF | `.pdf` |

- Расширение выходного файла обновляется автоматически при смене формата
- Формат входного файла определяется автоматически по расширению

### Общее

- Тёмная тема
- UTF-8 везде — кириллица и другие не-латинские символы работают без дополнительных настроек
- Конвертация выполняется в фоновом потоке — интерфейс не зависает
- MarkItDown и Pandoc устанавливаются автоматически при первом запуске

## Требования

- Python 3.10 или новее
- Интернет-соединение при первом запуске (для загрузки бинарника Pandoc)
- Для вывода в PDF: LaTeX-движок, например [MiKTeX](https://miktex.org/) (Windows) или TeX Live (Linux/macOS)

## Установка

```bash
git clone https://github.com/Lueur3/docforge.git
cd docforge
pip install -r requirements.txt
python main.py
```

При первом запуске MarkItDown и Pandoc загружаются и настраиваются автоматически.
Ручная установка внешних инструментов не требуется.

---

## Опционально: ffmpeg

ffmpeg включает конвертацию аудио и видео файлов во вкладке MarkItDown
(например, извлечение текста из `.mp3`, `.mp4`, `.wav`).

**Установка:** нажмите кнопку **Установить ffmpeg** во вкладке MarkItDown.
Приложение использует [imageio-ffmpeg](https://github.com/imageio/imageio-ffmpeg) — ручная загрузка не нужна.

Если ffmpeg уже установлен в системном PATH, он будет обнаружен автоматически.

## Использование

**Вкладка MarkItDown**

1. Нажмите **Обзор** и выберите входной файл
2. Путь к выходному файлу подставится автоматически
3. При необходимости скорректируйте путь
4. Нажмите **Конвертировать**

**Вкладка Pandoc**

1. Нажмите **Обзор** и выберите входной файл
2. Выберите формат вывода из выпадающего списка
3. Путь к выходному файлу обновится автоматически — скорректируйте при необходимости
4. Нажмите **Конвертировать**

Результаты конвертации и ошибки отображаются в области лога в нижней части каждой вкладки.
